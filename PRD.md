# 2026 世界杯竞猜游戏 — PRD v1.1

## 产品概述

面向小团队的 2026 FIFA World Cup 虚拟竞猜游戏。注册送 $100 虚拟货币，Polymarket 风格动态赔率下注。不涉及真实交易。

## 已确认决策

| 决策 | 结论 |
|------|------|
| 管理员 | 固定 admin 账号（admin/admin123） |
| 破产机制 | 亏光后只能观战，不补充 |
| 盘口类型 | 1X2胜负 + 大小球(O/U 2.5) + 正确比分 |
| 测试数据 | 2024欧洲杯淘汰赛 |
| 结算方式 | 与 Polymarket 同步（正确选项 $1/股，错误归零） |
| UI风格 | 足球主题、简洁清晰、带描述和预计收益 |

## 盘口详情

### 1. 1X2 胜负
- 选项：主胜 / 平局 / 客胜
- 结算：按比赛结果（含加时/点球）

### 2. 大小球 (Over/Under 2.5)
- 选项：大于 2.5 球 / 小于等于 2.5 球
- 结算：按总进球数

### 3. 正确比分
- 选项：0:0, 1:0, 0:1, 1:1, 2:0, 0:2, 2:1, 1:2, 2:2, 其他
- 结算：精确匹配最终比分

## Polymarket 下注机制

- 每个盘口选项是一个"股票"
- 价格 = 该选项总下注 / 盘口总下注（$0.01 ~ $0.99）
- 买入：花费 $X，获得 shares = $X / price
- 结算：正确选项每股 = $1，错误 = $0
- 下注截止：比赛开赛时间
- 每人每个盘口只能下注一次
